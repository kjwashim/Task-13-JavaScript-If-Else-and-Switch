# Even or Odd Checker

A simple JavaScript program that checks whether a given number is **even** or **odd** using an `if-else` statement.

## Objective

The objective of this assignment is to create a JavaScript program that determines whether a given number is even or odd.

## Technologies Used

- HTML5
- JavaScript

## Project Structure

```text
Even-Odd-Checker/
│── index.html
│── script.js
└── README.md
```

## Features

- Uses four different variables.
- Checks each number using an `if-else` statement.
- Displays the result in the browser console using `console.log()`.

## How to Run

1. Clone or download the project.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Press **F12** and open the **Console** tab.

## Sample Output

```text
The number 5 is odd.
The number 10 is even.
The number 15 is odd.
The number 20 is even.
```
